# Contributing

Choose the repository that owns the next action:

- submit source leads and reconnaissance to `MATHFORGE`;
- submit work packages, reductions, and failure analysis to `MATHSOLVE`;
- submit formalizations and replayable certificates to `MATHCERT`;
- submit standards and governance changes to `MATH-PROGRAMME`.

Use the repository issue forms. Cross-pillar submissions must include an immutable
upstream commit SHA, a stable identifier, and an explicit promotion condition.
